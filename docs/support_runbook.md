# Support Runbook

## Common Issue: Request status not updating
Check:
- Request ID exists
- Status value is valid
- User has correct role
- PL/SQL package compiled successfully

## Common Issue: KPI report missing data
Check:
- Request exists in business_process_requests
- Metric exists in process_kpi_metrics
- Reporting view compiled successfully

## Common Issue: Approval history missing
Check:
- Approver ID exists
- Request ID exists
- approve_request procedure completed successfully

## Support Notes
This runbook simulates production support documentation used in enterprise application support and Oracle package implementation environments.
